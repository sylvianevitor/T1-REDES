interação cli/servidor CGI
sockets em python
criar aplicativo servidor multi-threaded

- permitir a um usuario realizar uma busca de resultados de linha de comando, a partir de um conjunto de “maquinas” Linux, atraves de uma interface web

requisicao: 
 -comandos linux: ps, df, finger ́e uptime
 -k maquinas de uma lista
resposta: 


cliente: front: pagina web em python em que usuario seleciona maquinas e comandos
servidor: back: conectar a conjunto de daemons rodando nas maquinas da lista, executar comandos nas maquinas, daemons executam localmente os comandos, redireciona a saida (pipe) para o back que junta as respostas e devolve pagina web com resultados

Programas:

webserver.py (um programa CGI acoplado a um servidor web apache) ou standalone

pagina HTML de formulario com os dados da requisicao (servida pelo programa webserver ou pelo servidor web apache + CGI)

backend.py - pode ser junto do webserver.py
https://docs.python.org/2/howto/sockets.html

daemon.py - executa comandos localmente, processa a saida usando IPV4
httpd
sshd
